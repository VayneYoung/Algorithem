## 剑指Offer53：正则表达式的匹配
####题目描述
请实现一个函数用来判断字符串是否表示数值（包括整数和小数）。例如，字符串"+100","5e2","-123","3.1416"和"-1E-16"都表示数值。 但是"12e","1a3.14","1.2.3","+-5"和"12e+4.3"都不是。  
####思路
考查正则表达式。  
```
//s字符串
function isNumeric(s)
{
    let reg = /^[+-]?\d*(\.\d+)?$/
    let reg2 = /^[+-]?\d+(\.\d+)?[Ee]{1}[+-]?\d+$/
    return s.match(reg)||s.match(reg2)
}
```
